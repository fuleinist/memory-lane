# MemoryLane

AI-powered CLI tool that automatically maintains a markdown journal of your coding work.

## Installation

```bash
npm install -g memory-lane
```

## Usage

### Initialize

```bash
memory-lane init
```

### Run the journaler

```bash
memory-lane run
```

### Search journal

```bash
memory-lane search <query>
```

## Configuration

Edit `.memorylanerc` after initialization to configure your LLM provider.
